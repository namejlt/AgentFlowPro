# AgentFlow Pro
**基于 Go + Vue3 构建的通用多智能体工作流编排与报告生成平台**

一款开源、可视化、可扩展的多智能体协作系统，支持自定义AI角色、可视化编排分析流程、接入多数据源、执行多轮讨论与验证，并自动生成全场景专业分析报告。

## 项目介绍
AgentFlow Pro 是一款面向通用场景的多智能体工作流平台，通过零代码/低代码方式，让用户快速构建AI分析工作流。平台支持自定义智能体、工作流编排、多模型管理、报告生成、历史记录、工作流分享等能力，可广泛应用于股票投研、高考志愿、商业分析、行业研究、项目评估、决策支持等各类专业场景。

平台采用前后端分离架构：Go 提供高性能后端服务，Vue3 提供现代化交互界面，开箱即用、易于部署、支持二次开发与私有化部署。

## 核心特性
- **智能体管理**：自定义角色、系统提示词、数据源、分析框架、输出格式
- **可视化工作流编排**：拖拽式设计，支持串行、并行、多轮讨论、辩论、投票、交叉验证
- **工作流全生命周期管理**：创建、编辑、复制、版本管理、分类标签、运行历史
- **工作流分享与互通**：一键导出、一键导入、一键分享，支持模板互通
- **LLM 配置中心**：多模型统一管理，支持 OpenAI、通义千问、文心一言、豆包等主流大模型
- **智能体绑定独立模型**：工作流中每个智能体可单独指定 LLM，灵活分配模型资源
- **报告运行管理**：历史报告列表、执行日志、详情查看、重新执行、导出、删除
- **多格式报告导出**：支持导出 Markdown、PDF、Word 等格式
- **多数据源支持**：HTTP API、文件上传、手动输入
- **全场景通用**：不绑定业务领域，一套平台适配所有分析场景
- **高性能与易用性**：Go 高并发后端 + Vue3 现代化前端，简洁高效

## 技术架构

- Web 框架：Gin
- 核心引擎：工作流执行引擎、智能体调度引擎、LLM 统一调用层、数据源引擎

- 流程图：Vue Flow / 拖拽式编排

## 典型应用场景
### 股票智能投研分析
- 智能体：技术分析师、基本面分析师、情绪分析师、风险控制专家
- 流程：多智能体并行分析 → 交叉验证 → 辩论汇总 → 生成专业投研报告
- 支持：工作流分享、历史报告回溯、多模型切换

### 高考志愿分析
- 智能体：分数评估师、院校分析师、专业规划师、就业顾问
- 流程：数据输入 → 智能匹配 → 风险提示 → 志愿方案报告
- 支持：一键导入志愿分析模板、历史方案管理

### 商业与行业分析
- 智能体：市场研究员、竞品分析师、财务分析师、战略顾问
- 流程：行业调研 → 数据分析 → 多轮评审 → 战略建议报告
- 支持：自定义分析框架、工作流模板共享

### 通用决策与评估
- 项目可行性分析、求职岗位评估、法律风险评估、教育规划、学术研究辅助等

## 快速开始
### 环境要求
- Go 1.21+
- Node.js 18+

### 后端启动
```bash
git clone https://github.com/namejlt/AgentFlowPro.git
cd AgentFlowPro/server
go mod tidy
go run main.go
```

### 前端启动
```bash
cd ../web
npm install
npm run dev
```

### 访问平台
- 前端页面：http://localhost:28232
- 后端接口：http://localhost:28131
